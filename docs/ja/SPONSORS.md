# ☕ サポート & 貢献

役に立ちましたか？作者にコーヒーをおごるか、WeChatグループに参加してサポートを受けてください。QRコードは[リポジトリトップ](README.md)にあります。

---

## 💖 プロジェクトをサポート

Gemini2APIは主に個人によって維持されているオープンソースプロジェクトです。このプロジェクトがお役に立った場合は、以下の方法でサポートをご検討ください：

- ⭐ プロジェクトにスターを付ける
- 🔗 必要としている友人と共有する
- 🐛 バグや提案を[Issues](https://github.com/xwteam/gemini2api/issues)で報告する
- 🔧 コードやドキュメントをPRで貢献する

---

## 📢 コミュニティグループ

WeChatグループに参加して、使用方法のヘルプ、経験の共有、新機能の議論を行ってください。[リポジトリトップ](README.md)のQRコードをスキャンして参加できます。

---

## 🤝 貢献する

1. このリポジトリをフォークする
2. ブランチを作成する `git checkout -b feature/your-feature`
3. コードをコミットする `git commit -m "feat: add something"`
4. プッシュしてプルリクエストを作成する

**貢献分野：**

- 🔑 Cookie長期保活ソリューション（現在約2時間で失効、経験豊富な開発者のアイデア貢献を歓迎）
- 🌐 多言語翻訳の改善
- 📱 モバイル適応の最適化
- 🧪 新しいモデルの互換性テスト
- 📖 ドキュメントの改善
